## 🚀 Cómo Desplegar

### **Opción 1: Netlify (Recomendado - GRATIS)**

1. **Subir a GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/tuusuario/posada-de-victor
   git push -u origin main
   ```

2. **Ir a netlify.com:**
   - Click en "New site from Git"
   - Conectar repositorio de GitHub
   - ¡Listo! Netlify despliega automáticamente

3. **Conectar dominio personalizado:**
   - En Netlify: Domain settings → Add custom domain
   - Apuntar DNS en Namecheap a Netlify

### **Opción 2: Drag & Drop en Netlify**
1. Ve a netlify.com
2. Arrastra la carpeta `posada-de-victor/` completa
3. Tu sitio se publica en segundos
4. Luego conectas el dominio

### **Opción 3: SiteGround (Hosting Tradicional)**
1. Sube los archivos vía FTP
2. Asegúrate que `index.html` esté en la raíz

## 📋 Checklist Antes de Subir

- ✅ El archivo principal es `index.html`
- ✅ Carpeta `assets/` está en el mismo nivel que `index.html`
- ✅ Todas las imágenes en `assets/images/`
- ✅ Video en `assets/videos/`
- ✅ Las rutas en HTML son relativas: `assets/images/posada-1.webp`
